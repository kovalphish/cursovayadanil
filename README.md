# ТехноМаркет - Интернет-магазин бытовой техники

## 🚀 Деплой на Vercel

### Подготовка к деплою:

1. **Установите Vercel CLI:**
```bash
npm i -g vercel
```

2. **Войдите в Vercel:**
```bash
vercel login
```

3. **Инициализируйте проект:**
```bash
vercel
```

4. **Настройте переменные окружения в Vercel Dashboard:**
   - `SECRET_KEY` - секретный ключ для Flask сессий
   - `DATABASE_URL` - URL базы данных (SQLite для разработки, PostgreSQL для продакшена)

### Переменные окружения:

```env
SECRET_KEY=your-super-secret-key-here
DATABASE_URL=sqlite:///database.db
```

### Деплой:

```bash
vercel --prod
```

## 📋 Функционал

### Клиентская часть:
- 🏠 Главная страница с популярными товарами
- 🛍️ Каталог товаров с поиском и фильтрацией по категориям
- 🛒 Корзина товаров
- 📦 Оформление заказа

### Админ-панель:
- 👨‍💼 Управление товарами (CRUD)
- 📊 Просмотр и управление заказами
- 🔄 Изменение статуса заказов
- 🖼️ Загрузка изображений товаров (хранятся в base64 в БД)

### Технологии:
- **Backend:** Flask + SQLAlchemy
- **Frontend:** HTML + CSS (встроенные стили)
- **База данных:** SQLite (для разработки) / PostgreSQL (для продакшена)
- **Изображения:** Хранятся в формате base64 в базе данных

## 🔧 Локальный запуск

1. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

2. **Запустите приложение:**
```bash
python app.py
```

3. **Откройте в браузере:**
```
http://127.0.0.1:5000
```
